## ğŸ› Troubleshooting

### Theme Not Changing?
1. Make sure you tapped the preset button completely
2. Check that the animated fade transition completes
3. Try toggling Custom Colors on/off
4. Try closing and reopening the app

### Filter Not Showing?
1. Make sure you've captured a photo first (tap the circle)
2. Check that the placeholder image loaded
3. Try selecting a different filter
4. Try adjusting the intensity slider

### Changes Not Persisting?
1. **Themes**: They should save automatically
   - Try closing and opening the app completely
   - Check phone storage isn't full

2. **Photos**: Currently doesn't persist (demo mode)
   - Future version with expo-camera will save photos

### App Crashing?
1. Try: `npm start` then reload the app (press `r`)
2. Try: Restart Expo Go app
3. Try: Clear app cache
4. Check: Console for error messages
